                              API ENDPOINTS
================================================================================

PROPERTYDATA API ENDPOINTS (Available Now):
-------------------------------------------
Base URL: https://api.propertydata.co.uk/

Relevant endpoints for floor plans/area:
- /floor-areas - Get floor area for a property
- /prices - Property price data
- /rents - Rental estimates
- /sold-prices - Historical sold prices
- /planning - Planning applications
- /epc - Energy Performance Certificate data

Authentication: API key in header
Documentation: https://propertydata.co.uk/api/documentation

EPC REGISTER API (To Be Integrated):
------------------------------------
Base URL: https://epc.opendatacommunities.org/api/v1/
Endpoints:
- /domestic/search - Search for EPC certificates
- /domestic/certificate/{lmk-key} - Get specific certificate

Data includes:
- Floor area (total and by floor)
- Property type
- Construction age
- Basic floor plan data (some certificates)
